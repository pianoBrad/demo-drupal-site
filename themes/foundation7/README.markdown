ZURB Foundation theme for Drupal 7
============================

This is a Drupal 7 theme built using Foundation 3.2.3 by Zurb - http://foundation.zurb.com

This version uses SCSS and Compass.


Regions
============================

* Topbar
* Header
* Navigation
* Left
* Right
* Content
* Footer
